## Quick setup

Stack: NestJS + Prisma + NeonDB + Swagger

```bash
# install dependencies
$ pnpm i

# start the dev server
$ pnpm run start:dev
```

Open your browser to [localhost:3000/api](localhost:3000/api)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
